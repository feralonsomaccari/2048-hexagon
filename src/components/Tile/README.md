# Tile Component

This is the game Tile component. It will render a Tile in the grid with a value

## Properties

| Property  | Type     | Default    | Description                |
| :-------- | -------- | :--------- | :------------------------- |
| **value** | `number` | `required` | Value of the tile.         |
| **style** | `object` | `{}`       | Inline styles of the tile. |
